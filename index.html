How to Read the DSS Decision
The dashboard prioritizes the latest live reading. Telegram notifications are sent only for irrigation and waterlogging conditions, while normal status remains visible in the dashboard to avoid unnecessary alerts.
Calibration Note
Thresholds should be refined using actual soil type, crop stage, root zone depth, rainfall condition, and field validation.

Simulation Guide

This section is for demonstration only. It explains which Wokwi sensor positions should trigger each DSS code and Telegram alert. The buttons are intentionally not connected to the sensors to avoid confusing the live monitoring workflow.

Agroforestry Test Cases
0
Soil moisture acceptable
Tree and crop readings are within the acceptable range. No Telegram alert is needed.
1
Tree zone irrigation recommended
Set tree zone dry while crop alley remains normal.
2
Crop alley irrigation recommended
Set crop alley dry while tree zone remains normal.
3
Both zones irrigation recommended
Set both tree zone and crop alley to dry condition.
4
Possible waterlogging
Set both zones to saturated condition.
Notification Logic
Telegram should only send alerts for DSS Codes 1, 2, 3, and 4. DSS Code 0 is intentionally shown only on the dashboard so the group chat will not be flooded with normal-status messages.

Recommended setup:
React Code 1 to Tree Telegram Alert
React Code 2 to Crop Telegram Alert
React Code 3 to Both Zones Telegram Alert
React Code 4 to Waterlogging Telegram Alert

Project Information

System layout prepared for a ridge-to-reef smart agriculture monitoring and decision-support platform.

Monitoring Components
The website separates the system into Agroforestry, AWD, and Rice-Fish monitoring sections to reduce data overload. Only the selected system is shown in detail, making the interface easier to understand during demonstrations.
Integration Status
Agroforestry soil moisture is currently connected to ThingSpeak. AWD water level and Rice-Fish water level and dissolved oxygen sections are prepared as layout modules and can be connected once the sensor fields are finalized.

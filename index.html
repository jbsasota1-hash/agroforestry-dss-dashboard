Project Information

System layout prepared for a ridge-to-reef smart agriculture monitoring and decision-support platform.
Monitoring Components
The website separates the system into agroforestry, AWD, and rice-fish monitoring sections to reduce data overload. Only the selected system is shown in detail, making the interface easier to explain and easier to use during demonstrations.
Integration Status
Agroforestry soil moisture is currently connected to ThingSpeak. AWD water level and rice-fish water level and dissolved oxygen sections are prepared as layout modules and can be connected later once the sensor fields are finalized.
